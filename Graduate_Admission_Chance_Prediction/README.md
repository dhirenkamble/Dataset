# Graduate Admission Chance Prediction

## Project Overview
This project focuses on predicting the probability of a student's admission
to a graduate program using academic and profile-based parameters.

## Parameters Used
- GRE Score
- TOEFL Score
- CGPA
- SOP
- LOR
- University Rating
- Research Experience

## Methodology
A simple Machine Learning approach using Linear Regression
is used to understand how different factors influence admission chances.
